# PicoCalc MMBasic Examples (Michael Adcock)

> Source: <https://michaeladcock.info/temp/PicoCalc_MMBasic.html>
> A TiddlyWiki by Michael Adcock indexing community MMBasic programs for PicoCalc / PicoMite / WebMite. Last update visible: 30 June 2025. Fetched 2026-05-17.

The site is downloadable as a single offline TiddlyWiki — useful for searching across all programs by keyword.

## Categories of programs

- **Art** (some animated, most procedurally generated)
- **Animations**
- **Applications**
- **Demos**
- **Fonts**
- **Games**
- **GPIO applications**
- **TURTLE Graphics**

It also points to:

- **ClockworkPi Forum** posts where MMBasic for PicoCalc is discussed.
- A list of **GitHub Repositories** containing MMBasic for PicoCalc.
- A linked **Style Guide for PicoCalc** (this is the same one Lennart Hennigs maintains in `LennartHennigs/PicoCalc-Notes`; mirrored locally in `references/community/mmbasic_style_guide.md`).

## Why this is useful for the course

- **Real working programs** that compile and run on the same RC23 firmware we have.
- A search-friendly index to find a working example of any keyword (e.g., `FRAMEBUFFER`, `PLAY TONE`, `SETTICK`).
- A few of these (mandelbrot, lissajous, lorenz, mand.bas) are already preloaded on this PicoCalc's `B:` drive — direct comparison points.

## How to use offline

The TiddlyWiki has a "download button" that saves the entire wiki — including all `.bas` source samples and screenshots — into a single HTML file. Worth grabbing before the site moves.
