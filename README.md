# mmBasic — мини-курс

Погружение в **MMBasic** на устройстве **ClockworkPi PicoCalc** (предустановленная прошивка `PicoMite MMBasic RP2040 Edition v6.00.02 RC23`).

## Для кого

Для того, кто когда-то писал на QBasic / школьном BASIC и хочет быстро освоить современный MMBasic, плюс по дороге разобраться с фишками PicoCalc — экраном 320×320, клавиатурой, звуком и SD-карточкой.

## Как пользоваться

Уроки идут по порядку — каждый рассчитан на 10–20 минут чтения и пару коротких сниппетов, которые набираются прямо на устройстве через `EDIT`. После каждого урока — мини-упражнение, чтобы пощупать материал руками.

## О языке: код на английском, комментарии на русском

Текст уроков на русском, но **строки в `PRINT` / `INPUT` и любой текст, который рендерится на LCD** — на английском. Причина простая: встроенные шрифты PicoMite (включая RC23 на PicoCalc) не содержат кириллических глифов, и попытка вывести «Привет» одной командой `PRINT` даст на экране псевдографику и греческие буквы — каждый байт UTF-8 рендерится как отдельный символ из латино-графического диапазона.

В **комментариях** (`'` или `REM`) кириллица работает свободно — токенизатор их игнорирует, на экран они не попадают. Поэтому в уроках:

```basic
' Считаем сумму очков
PRINT "Total score: "; total
```

Подробный разбор «почему так» и сценарии локализации (свой CP1251-шрифт через `DefineFont`, рендер русских надписей через `LOAD IMAGE`) — в `references/picocalc/russian_locale.md`. Если у тебя есть PicoCalc и хочется русский на экране — это решаемо, но это отдельный путь, не в фокусе этого вводного курса.
